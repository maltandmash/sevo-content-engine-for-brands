# Drafts

Finished draft packs are written to your brand folder under `drafts/<slug>/`, not here. This folder inside the plugin is only a placeholder, because the plugin is replaced on update and your work should not live inside it.

Each piece is one folder:

```
drafts/<slug>/
├── article.md            the main piece, with frontmatter and a linked Sources section
├── repurposing-pack.md   cross-platform assets (Reddit, LinkedIn, YouTube, social)
└── review-notes.md       fact-check log, [VERIFY] flags, compliance report, what to test next
```

Read `review-notes.md` first. For regulated sectors a draft is not finished copy until a qualified reviewer has signed it off.
